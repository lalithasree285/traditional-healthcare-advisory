# Traditional Healthcare Advisory

A web-based tool that helps users identify potential conditions based on symptoms, using fuzzy matching to handle typos and varied phrasing.

## Features
- Symptom-based search
- Fuzzy matching for flexible input
- About page with project details

## Tech Stack
- HTML, CSS, JavaScript

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
